# ⚖️ Disciplinary Board Management Application

> **University of Jendouba** · Full-Stack Web Application · 2022/2023

A web application to streamline the management of student disciplinary cases at the University of Jendouba.

---

## 📌 Overview

| Role | Capabilities |
|------|-------------|
| 🛠️ **Admin** | Manage cases, members, sanctions, meetings |
| 👨‍💼 **Council Member** | Review cases, communicate, track decisions |
| 🎓 **Student** | View case status and decisions |

---

## 🚀 Features

- ✅ Secure storage and tracking of disciplinary case documentation
- ✅ Communication tools between administrative staff and council members
- ✅ Full traceability of all decisions and actions
- ✅ Sanction and meeting management
- ✅ Role-based access control

## ▶️ How to Run

**Backend:**
```bash
cd backend
npm install
node server.js
```

**Frontend:**
```bash
cd project/CrudBasico
npm install
ng serve
```
